# Pokédex Interactiva

Una aplicación web interactiva para explorar el mundo de los Pokémon.

## Características

- Visualización de Pokémon con información detallada
- Búsqueda por nombre o número
- Filtrado por tipos
- Diseño responsive
- Interfaz moderna y atractiva

## Estructura del Proyecto
pokemon-pokedex/
├── index.html
├── css/
│ ├── style.css
│ └── responsive.css
├── js/
│ ├── app.js
│ ├── pokemon-data.js
│ └── ui.js
└── README.md

## Cómo usar

1. Clona o descarga el proyecto
2. Abre `index.html` en tu navegador
3. Explora los Pokémon disponibles
4. Usa la barra de búsqueda para encontrar Pokémon específicos
5. Filtra por tipo usando los botones
6. Haz clic en cualquier Pokémon para ver sus detalles

## Tecnologías

- HTML5
- CSS3 (Variables CSS, Grid, Flexbox)
- JavaScript Vanilla
- Font Awesome para iconos
